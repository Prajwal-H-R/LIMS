# Certificate Assets

Place the following image files in this folder for certificates to display correctly:

- **logo_left.png** – Left header logo
- **logo_right.png** – Right header logo  
- **htw_qr_code_certificate.png** – QR code shown in certificate notes

These images are served via `/api/certificate-assets/` and used in both preview and PDF generation. If a file is missing, a transparent placeholder will be shown instead.
